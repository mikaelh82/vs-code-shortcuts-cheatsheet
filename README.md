# Visual Studio Code Shortcuts

Following is a list of useful shortcuts for Visual Studio Code that I am using personally on a daily basis on my Mac. This guide uses `Cmd` (⌘), `Ctrl` (^), `Alt` (⌥), `Shift` (⇧) as the primary modifier keys.

## Code

| Shortcut                        | Description                                    |
|---------------------------------|------------------------------------------------|
| `Ctrl + Space` / `Alt + Tab`    | Trigger IntelliSense suggestions               |
| `Shift + Cmd + Space`           | Trigger parameter hints (inside function)      |
| `Alt + F12`                     | Peek definition                                |
| `Cmd + .`                       | Quick fix                                      |
| `Shift + Cmd + M`               | Show problems                                  |
| `Shift + Cmd + M` → Quick fix → Fix with Copilot | Show problems and quick fix with Copilot     |
| `Cmd + K, Cmd + I`              | Show hover function information                |
| `F8`                            | Go to previous error/warning                   |
| `Ctrl + Shift + \`              | Toggle terminal                                |

## Window Management

| Shortcut                        | Description                       |
|---------------------------------|-----------------------------------|
| `Cmd + B`                       | Toggle sidebar visibility         |
| `Ctrl + Shift + Alt + Cmd + 7`  | Split window                      |
| `Cmd + K Z`                     | Zen mode                          |
| `Esc Esc`                       | Exit Zen mode                     |
| `Cmd + Number`                  | Go to window x (replace Number with a digit) |
| `Cmd + W`                       | Close window                      |
| `Cmd + Ctrl + F`                | Toggle full screen                |
| `Shift + Cmd + E`               | Toggle explorer                   |
| `Cmd + Shift + P`               | Show command palette              |

## Code Navigation

| Shortcut                        | Description                         |
|---------------------------------|-------------------------------------|
| `Shift + Alt + F12`             | Show references                     |
| `F12`                           | Go to definition                    |
| `Cmd + T`                       | Show all symbols                    |
| `Shift + Cmd + O`               | Go to symbol                        |
| `Ctrl + G`                      | Go to line                          |
| `Cmd + P`                       | Go to file                          |
| `Home`                          | Go to beginning of line             |
| `End`                           | Go to end of line                   |
| `Cmd + Arrow Up`                | Go to beginning of file             |
| `Cmd + Arrow Down`              | Go to end of file                   |
| `Ctrl + Page Up`                | Scroll line up                      |
| `Ctrl + Page Down`              | Scroll line down                    |
| `Cmd + Page Up`                 | Scroll page up                      |
| `Cmd + Page Down`               | Scroll page down                    |

## Editing

| Shortcut                        | Description                         |
|---------------------------------|-------------------------------------|
| `F2`                            | Rename symbol                       |
| `Cmd + X`                       | Cut line                            |
| `Cmd + C`                       | Copy line                           |
| `Alt + Arrow Up/Down`           | Move line up/down                   |
| `Shift + Alt + Arrow Up/Down`   | Copy line/selection up/down         |
| `Shift + Cmd + K`               | Delete line/selection               |
| `Cmd + Enter / Shift + Cmd + Enter` | Insert line below/above         |
| `Ctrl + Cmd + Alt + 9`          | Indent right                        |
| `Ctrl + Cmd + Alt + 8`          | Indent left                         |
| `Cmd + K Cmd + C`               | Add comment                         |
| `Cmd + K Cmd + U`               | Remove comment                      |
| `Cmd + Shift + 7`               | Toggle line comment                 |

## Multiline Cursors

| Shortcut                        | Description                                        |
|---------------------------------|----------------------------------------------------|
| `Alt + Click`                   | Insert cursors                                     |
| `Alt + Cmd + Arrow Up/Down`     | Insert cursor above/below                          |
| `Cmd + L`                       | Select current line                                |
| `Shift + Alt + I`               | Insert cursor at the end of each line selected     |
| `Shift + Cmd + L`               | Select all occurrences of the current selection    |
| `Cmd + F2`                      | Select occurrences of current word                 |
